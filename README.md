## 💼 실무 프로젝트

회사에서 진행한 프로젝트입니다. 사내·고객사 코드라 저장소는 비공개여서 내용 위주로 적었습니다.

### 도면 통합 관리 시스템 (PLM)
조선·해사 분야 고객사에 납품하는 PLM 솔루션을 커스터마이징했습니다. 제가 맡은 부분은 다음과 같습니다.

- WBS 간트 화면을 팝업으로 개편하고, 엑셀로 일정을 한 번에 등록하는 기능을 만들었습니다.
- 진도율 관리 화면을 개발했습니다. 프로젝트 도면을 자동으로 불러오고, 가중평균으로 진행률을 계산하고, PM과 설계자 권한을 나눴습니다.
- 도면 배포 관리에서 재배포 시 리비전 이력을 남기고, 배포 예정일 5일 전에 알림이 뜨게 했습니다.
- 운영 중 생긴 DB 문제(계정 인증, 한글 깨짐, sql_mode 등)도 직접 잡았습니다.

기술: Spring Boot, Thymeleaf, DevExtreme, MariaDB, SVN

### 제조 생산관리 HMI 대시보드
제조 고객사의 생산 현황을 보여주는 화면을 개발했습니다. 대시보드 UI와 작업지시서 처리 로직, 도면 조회 기능을 맡았습니다.

기술: React

### 악취 모니터링 대시보드
현장 센서가 측정한 악취 데이터를 SCADA로 모으고, 서버(WAS)에 쌓아서 대시보드로 보여주는 시스템을 만들어 납품했습니다. 배포는 Docker Compose로 했습니다.

기술: Spring Boot, JSP, SCADA, Docker Compose
